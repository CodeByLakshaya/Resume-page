# Single Page Portfolio Website

## Overview

This project is a **single-page personal portfolio/resume website** designed to showcase professional experience, skills, projects, and education in a clean and structured layout. It is suitable for developers who want a simple, lightweight, and easily deployable personal webpage.

The page is fully static and can be hosted on any basic web server or platform such as GitHub Pages, Netlify, or Vercel.

---

## Features

* Clean, professional resume-style layout
* Clearly structured sections:

  * About Me
  * Skills
  * Experience
  * Projects
  * Education
  * Contact Information
* Easy to customize and extend
* No external dependencies required

---

## Technologies Used

* **HTML5** – Page structure and semantic layout
* **CSS3** – Styling and visual formatting

---

## Sections Breakdown

### Header

Displays the name and professional title at the top of the page.

### About Me

A short professional summary highlighting experience and technical focus.

### Skills

A bullet list of core technical skills, including front-end, back-end, and tooling.

### Experience

A table-based layout showing:

* Job title
* Company name
* Duration of employment

### Projects

Detailed descriptions of key projects, including:

* Project purpose
* Technologies used
* Development timeline

### Education

Academic background and certifications displayed in a table format.

### Contact

Essential contact details such as:

* Email address
* Phone number
* Location

## How to Use

1. Clone or download the project files.
2. Open the main HTML file in any modern web browser.
3. Edit the HTML content to replace placeholder information with your own details.
4. Customize the CSS as needed for branding or layout changes.

## Deployment

You can deploy this page using:

* GitHub Pages

* Vercel
* Any standard web hosting service
